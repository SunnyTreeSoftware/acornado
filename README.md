Photo Profile Management System for Laravel  

## Installation  
Install Laravel  
`
laravel new acorn
`  

Laravel Auth  
`
php artisan make:auth  
`

Migrations  
`
php artisan migrate  
`  

Install Acorn  
`
composer require sunnytreesoftware/acorn
`  

Publish Assets  
`
php artisan vendor:publish --provider="Sunnytree\Acorn\AcornServiceProvider"
`
